# WebSocket NodeJS Chat

Sample project created to test WebSocket API using TDD and NodeJS.

To install:
* check out this repo
* `npm install`
* `bower install`
* `grunt production`

To run the app:
* `node run-server.js`
* open index.html in the browser

To run tests:
* `grunt testMocha`
