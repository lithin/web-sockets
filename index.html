WebSockets Playground

This is a simple chat room using web sockets, created as a very basic playground for trying out the technology. It turned out developing with WS is fun, straight-forward but rather difficult to write proper unit tests for.

You can read more about it here: My Web Sockets Experience

Wonder how exactly it works? See the source code on github.
